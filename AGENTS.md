# Пайплайн разработки электронных курсов с AI-агентами

## Структура
* `sources/` — исходники.
* `workspace/` — бриф, структура, отчёты.
* `content/` — материалы курса (YAML-страницы, медиа).
* `project-state.json` — статус проекта.

## Основные этапы
1. **Анализ:** конвертация файлов, классификация курса, сборка брифа.
2. **Проектирование:** создание матрицы курса и шаблонов страниц.
3. **Генерация:** генерация содержания.
4. **Проверки качества:** аудит фактов, логики и техники, затем исправления.
*Дополнительно:* Ручные правки контента по запросу в любой момент.

## Правила
* Подробные правила этапов находятся в соответствующих skill-файлах.
* Всегда начинай с чтения `project-state.json` и текущих артефактов проекта.
* Не пропускай этапы без явного запроса пользователя.
* Изменяй только те файлы, которые относятся к текущей задаче.
