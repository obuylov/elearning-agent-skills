# Бриф проекта: Основы информационной безопасности

## --- Compliance Analysis ---

```yaml
type: compliance
focus: Соблюдение требований политики ИБ и снижение рисков кибератак
behavior_change_required: true
key_risks:
  - Фишинговые атаки на сотрудников (12 инцидентов за 2025)
  - Компрометация учётных записей из-за слабых паролей
  - Утечка конфиденциальной информации через email
  - Несоблюдение политики чистого стола
regulatory_references:
  - Политика ИБ ПАО «ТехноГрупп» (ред. 01.2026)
  - ФЗ-152 «О персональных данных»
  - Стандарт ISO 27001
certification:
  required: true
  type: Допуск к работе с корпоративными информационными системами
  passing_score: 80%
  recertification_period: 12 месяцев
assessment_strategy: >-
  Жёсткий контроль знаний. Пороговый балл 80% для допуска.
  Обязательная повторная сертификация через 12 месяцев.
  Непрошедшие — повторное прохождение через 3 дня.
design_principles:
  - Каждое требование подкреплено примером нарушения и последствий
  - Тестирование на понимание, а не на запоминание формулировок
  - Минимум «воды» — только релевантный контент
  - Реальные кейсы из практики Компании
```
