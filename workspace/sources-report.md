# Аудит исходных материалов

## Дата аудита: 2026-03-20

## Общая оценка

Исходные материалы покрывают две ключевые темы: общую политику ИБ компании и конкретные угрозы (фишинг, социальная инженерия). Материалы актуальны, но написаны сухим юридическим языком — потребуется серьёзная адаптация для формата электронного курса.

## Файл: word.docx → word.md

| Параметр | Значение |
|----------|----------|
| Объём | ~3 200 слов |
| Актуальность | Актуален (редакция от 01.2026) |
| Полнота | Покрывает 80% нужных тем |
| Качество | Юридический язык, требует упрощения |

**Сильные стороны:**
- Полный перечень требований к паролям
- Чёткая классификация информации по уровням конфиденциальности
- Описание процедуры реагирования на инциденты

**Слабые стороны:**
- Нет примеров из практики
- Отсутствуют визуальные материалы
- Язык слишком формальный для обучения

## Файл: myfiles.pdf → myfiles.md

| Параметр | Значение |
|----------|----------|
| Объём | ~1 800 слов |
| Актуальность | Актуален (редакция от 02.2026) |
| Полнота | Хорошо раскрывает тему фишинга |
| Качество | Более живой язык, есть примеры |

**Сильные стороны:**
- Реальные примеры фишинговых писем
- Пошаговая инструкция «Что делать при подозрении»
- Статистика инцидентов за 2025 год

**Слабые стороны:**
- Не описаны вишинг и смишинг (голосовой и SMS-фишинг)
- Нет интерактивных элементов — только текст
