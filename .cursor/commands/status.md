Покажи текущее состояние проекта.

## Действие

1. Прочитай [project-state.json](../../project-state.json).
2. Выведи компактную сводку:

```
═══ Статус проекта ═══

Режим исходников: {source_mode}

Этапы:
  ✅ start              — выполнен
  ✅ analysis-scanner   — выполнен
  ✅ analysis-router    — выполнен
  ✅ analysis-general   — выполнен
  ✅ course-structure   — выполнен
  ⬜ course-matrix      — не начат       ← следующий
  ⬜ course-template    — не начат
  ⬜ generation-text    — не начат
  ⬜ generation-quiz    — не начат
  ⬜ generation-image   — не начат
  ⬜ quality-audit      — не начат
```

3. Если есть `_active` — покажи:
   > ⚙️ Сейчас выполняется: {skill}, прогресс: {pages_done}/{pages_total}

4. Если есть страницы с `_lock` — покажи количество заблокированных страниц.

5. Кратко скажи, какой шаг следующий и какой командой его запустить.

## Правила

- Не меняй никакие файлы. Это чисто читающая команда.
- Показывай только реальные данные из [project-state.json](../../project-state.json).
- Используй ✅ для выполненных, ⬜ для невыполненных, ⚙️ для текущего активного.
